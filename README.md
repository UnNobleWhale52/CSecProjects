## ğŸ¤ How to Contribute

We welcome contributions! Here's how to add your own guide:

### Quick Start

1. **Fork this repository**
2. **Copy the template:**
   ```bash
   cp guides/TEMPLATE.html guides/XX-your-guide-name.html
   ```
3. **Edit your guide** - replace placeholder content with your knowledge
4. **Update index.html** - add a link to your new guide
5. **Submit a Pull Request**

### Writing Guidelines

- **Focus on readability** - use clear language and logical structure
- **Be practical** - include real commands and examples
- **Explain the "why"** - help readers understand, not just copy commands
- **Use the template** - maintain consistent styling across all guides
- **Include exercises** - give readers something to practice

### Template Features

The template includes pre-styled components:
- Code blocks with syntax highlighting
- Info boxes (tip, warning, note)
- Tables
- Navigation links
- Responsive design
